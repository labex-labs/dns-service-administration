Aprenda Administração de Serviços DNS. Este módulo abrange Instalação do BIND9, Configuração do Resolvedor de Cache (Caching Resolver), Criação de uma Zona Mestra (Master Zone), Gerenciamento de Registros DNS e Ferramentas de Solução de Problemas de DNS (DNS Troubleshooting Tools). Você dominará essas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
